CODAP Workshop Test

Last updated 3/14/23

1. Introduction/CODAP tutorials (10 min)

Workshop participants start by breaking into pairs/small groups to run through the two getting started tutorials (tutorial 1 link; tutorial 2 link—also embedded below). Participants work together in pairs/small groups to help one another if they get stuck.

2. Group discussion (10 min)

  • What did you see in the tutorials that are things CODAP allows you to do with data?

  • End with brief overview of background of CODAP, and showing the CODAP website/help resources on it (point out help system, forum, launch CODAP button, and example CODAP documents)

3. Exploring CODAP example documents (10 min)

  • Link to CODAP example documents (also embedded below)

  • Each group/pair of participants picks a dataset of interest to them (could be math, science, social science, language arts, or others)

  • Participants experiment with what to do with the dataset in CODAP (creating graphs, maps, etc.)

  • Goal is to get participants exploring data in CODAP, learn some things they can do, find some interesting patterns in the data, and start thinking about how they could use CODAP in their work (not to learn all the features of CODAP)

  • Ask participants to capture questions and challenges that come up during their exploration

4. Group discussion (10 min)

  • What did you learn from exploring the data in CODAP?

  • How could you use what you have learned about CODAP/expand on it in your own work?

    • Immediate-term: Find a simple, quick/easy thing you could do with students aligned with your current curriculum tomorrow

    • Longer term: What are you doing in your work that CODAP could help illustrate?

  • Emphasize that there is no need to learn everything CODAP can do all at once or completely reconfigure your curriculum for CODAP

5. Conclusion (10 min)

  • Emphasize useful resources for continued CODAP learning/work going forward

  • Have participants use the help system to seek an answer to one of the questions or challenges that came up during their exploration

  • Point out the following resources:

    • CODAP challenge cards (incl. printable version)

    • CODAP website

    • CODAP tutorials:

      • Getting started CODAP tutorials: tutorial 1 link; tutorial 2 link

      • CIDSEE project CODAP tutorial link

      • NetApp project tutorials: Mini-tutorial #1; Mini-tutorial #2; Mini-tutorial #3; Mini-tutorial #4

    • Tim Erickson’s Awash in Data book and codap.xyz website

    • Pip Arnold’s CODAP resources, etc.
